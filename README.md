# House Price Prediction API 🏠

This project is a **Machine Learning based House Price Prediction system** developed using Python.  
It trains a regression model using housing data and deploys it as a **REST API using Flask**.  
Users can send house features such as area, bedrooms, bathrooms, and location to get the predicted house price.

---

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Model Training
4. Model Serialization (Pickle)
5. API Development using Flask
6. API Testing using Thunder Client / Postman
